## What is Lynx Note?

Lynx Note is a modern, collaborative note-taking web application built with Ruby on Rails. Inspired by Notion, it provides a powerful yet intuitive platform for creating, organizing, and sharing your notes with friends and colleagues.

<Note>
Lynx Note combines the simplicity of traditional note-taking with advanced features like collections, real-time sharing, and friendship-based collaboration.
</Note>

## Core Concepts

### Notes

Notes are the fundamental building blocks of Lynx Note. Each note can contain:

- A title for quick identification
- Multiple content blocks that can be:
  - **Text**: Rich text paragraphs
  - **Lists**: Semicolon-separated list items
  - **Images**: Uploaded files stored securely

```ruby
# Note model structure
class Note
  field :title, type: String
  field :content, type: Array
  
  belongs_to :user
  has_many :sharings, as: :shareable
end
```

### Collections

Collections help you organize related notes together. Each collection:

- Has a descriptive name
- Contains an array of note IDs
- Can be shared with friends as a complete unit
- Automatically updates when notes are added or removed

```ruby
# Collection model structure
class Collection
  field :name, type: String
  field :notes, type: Array, default: []
  
  belongs_to :user
  has_many :sharings, as: :shareable
end
```

### Friendships

The friendship system enables collaboration through a request-based workflow:

1. **Send Request**: Initiate a friendship with another user
2. **Pending Status**: Request awaits acceptance
3. **Accept/Decline**: Recipient chooses to accept or reject
4. **Active Friendship**: Once accepted, both users can share content

```ruby
# Friendship workflow
class Friendship
  field :status, type: String, default: "pending"
  
  belongs_to :user, inverse_of: :initiated_friendships
  belongs_to :friend, class_name: 'User'
end
```

### Sharing

Sharing is implemented through a polymorphic model that works with both notes and collections:

- Share with multiple friends simultaneously
- Update sharing permissions at any time
- Remove access by unsharing
- Notifications keep everyone informed

<Warning>
When you share a note or collection, your friends gain the ability to view and edit the content. Make sure you trust the people you share with.
</Warning>

## Technology Stack

Lynx Note is built with modern, proven technologies:

- **Ruby on Rails 7.1.3**: Robust MVC framework
- **MongoDB with Mongoid**: Flexible NoSQL database for document storage
- **BCrypt**: Secure password hashing and authentication
- **CarrierWave**: File upload handling for images
- **Turbo & Stimulus**: Modern frontend interactivity
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Architecture Overview

Lynx Note follows Rails conventions with a MongoDB backend:

```
app/
├── models/
│   ├── user.rb           # User authentication and relationships
│   ├── note.rb           # Note content and validation
│   ├── collection.rb     # Note organization
│   ├── friendship.rb     # User connections
│   └── sharing.rb        # Polymorphic sharing model
├── controllers/
│   ├── notes_controller.rb
│   ├── collections_controller.rb
│   ├── friendships_controller.rb
│   └── sharing_controller.rb
└── views/
    # Responsive views for all resources
```

## Security Features

- **Password Encryption**: All passwords are hashed using BCrypt before storage
- **Authentication Required**: Most actions require user login
- **Authorization Checks**: Users can only access their own content or shared content
- **Email Validation**: Ensures valid email formats during registration
- **Secure File Uploads**: Images are stored with unique UUIDs
