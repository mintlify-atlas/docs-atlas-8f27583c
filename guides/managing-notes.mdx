## Overview

Lynx Note provides a flexible note management system that supports multiple content types, including text, lists, and file attachments. This guide covers everything you need to know about creating, editing, and organizing your notes.

## Creating a New Note

<Steps>
  <Step title="Access Note Creation">
    From your notes dashboard, click the "New Note" or "Create Note" button to open the note editor.
  </Step>
  
  <Step title="Add a Title">
    Enter a descriptive title for your note. This is required and helps you identify notes quickly in your dashboard.
  </Step>
  
  <Step title="Add Content">
    Build your note using one or more content types:
    - **Text blocks**: For paragraphs, headings, and general content
    - **Lists**: For to-do items, checklists, or bullet points
    - **File attachments**: For images and documents
  </Step>
  
  <Step title="Assign to Collections (Optional)">
    Select one or more collections to organize your note. You can add the note to collections you own or collections that have been shared with you.
  </Step>
  
  <Step title="Save Your Note">
    Click "Create Note" to save. You'll be redirected to view your new note with a confirmation message.
  </Step>
</Steps>

<Note>
  On mobile devices, you'll be redirected directly to the note view after creation. On desktop, you'll return to the notes dashboard with your new note highlighted.
</Note>

## Working with Content Types

### Text Content

Text blocks are the foundation of your notes:

- Add as many text blocks as needed throughout your note
- Each text block is stored and managed independently
- Empty text blocks are automatically removed when saving
- Use text blocks for paragraphs, descriptions, and general information

### Lists

Create organized lists for tasks, items, or structured information:

- Add list items one at a time
- Empty list items are automatically filtered out when saving
- List items are stored efficiently with semicolon separators
- Perfect for to-do lists, shopping lists, or step-by-step instructions

<Tip>
  Lists in Lynx Note are flexible - you can add as many items as you need, and the system will automatically clean up any blank entries when you save.
</Tip>

### File Attachments

Enhance your notes with images and files:

- Upload files directly into your note content
- Each file is assigned a unique identifier for security
- Supported file types include images and documents
- Files are stored securely and only accessible to you and users you share with

<Note>
  When you upload a file, it's processed and stored with a unique filename to prevent conflicts and enhance security.
</Note>

## Viewing Your Notes

Access and review your notes easily:

- View all your notes from the main notes dashboard
- Click any note to see its full content
- See which notes are shared and with whom
- Notes shared with you by friends are also accessible from your dashboard

### Shared Note Indicators

When viewing a note, you'll see visual indicators if:
- The note is shared with friends
- The note was shared with you by someone else
- Which specific friends have access

## Editing Notes

<Steps>
  <Step title="Open the Note">
    Navigate to the note you want to edit and click the "Edit" button.
  </Step>
  
  <Step title="Modify Content">
    Update any aspect of your note:
    - Change the title
    - Edit existing text blocks
    - Add or remove list items
    - Upload new files or remove existing ones
    - Update collection assignments
  </Step>
  
  <Step title="Manage Attachments">
    For files and images:
    - Replace existing files by uploading new ones
    - Mark files for deletion if no longer needed
    - Add new file attachments in any position
  </Step>
  
  <Step title="Save Changes">
    Click "Update Note" to save your modifications. You'll see a confirmation message and be redirected based on your device type and sharing status.
  </Step>
</Steps>

<Tip>
  When editing shared notes, your changes are visible to all users who have access. Be mindful when editing notes shared with friends or those shared with you.
</Tip>

### Managing Files During Edits

Lynx Note provides fine-grained control over file attachments:

- **Replace files**: Upload a new file in the same position to replace an existing one
- **Delete files**: Mark files for deletion, and they'll be removed when you save
- **Add new files**: Insert new file attachments anywhere in your note
- **Automatic cleanup**: Deleted files are permanently removed from storage to save space

## Organizing with Collections

Connect your notes to collections for better organization:

- Assign notes to multiple collections during creation or editing
- Add notes to your own collections or shared collections you have access to
- The note editor shows which collections currently contain the note
- Changes to collection assignments are saved with the note

<Note>
  You can add notes to collections that have been shared with you, making it easy to collaborate and organize shared content.
</Note>

## Deleting Notes

When you no longer need a note:

<Steps>
  <Step title="Select the Note">
    Navigate to the note you want to delete. You can only delete notes you created.
  </Step>
  
  <Step title="Confirm Deletion">
    Click the "Delete" button and confirm your decision.
  </Step>
  
  <Step title="Automatic Cleanup">
    The note is permanently removed along with:
    - All file attachments (deleted from storage)
    - Collection associations
    - Sharing relationships
  </Step>
</Steps>

<Note>
  When you delete a note, all associated files are automatically removed from storage. This action cannot be undone, so make sure you've saved any important content elsewhere.
</Note>

## Permissions and Access Control

### Your Own Notes
- Full edit and delete permissions
- Can share with friends
- Can add to any collection you have access to

### Notes Shared With You
- Can view and edit content
- Can add to collections
- Cannot delete the note (only the owner can)
- Changes are visible to the owner and other shared users

### Notes You Cannot Access
- Notes owned by others that haven't been shared with you
- Attempting to access these will redirect you to the dashboard

## Best Practices

<Tip>
  Create descriptive titles for your notes to make them easier to find later. Consider using consistent naming conventions like "Meeting Notes - [Date]" or "Project: [Name]".
</Tip>

- **Organize as you go**: Assign notes to collections immediately to stay organized
- **Use appropriate content types**: Text for long-form content, lists for structured items
- **Clean up regularly**: Delete notes you no longer need to keep your dashboard manageable
- **Leverage collections**: Group related notes together for easier navigation
- **Mind the sharing**: Remember that edits to shared notes affect all users with access

## Device-Specific Behavior

### Mobile Experience
- After creating or editing notes, you're taken directly to the note view
- Optimized interface for touch interactions
- Streamlined navigation for smaller screens

### Desktop Experience
- After creating or editing notes, you return to the dashboard with the note highlighted
- Full-featured interface with more screen real estate
- Better for managing multiple notes and collections simultaneously
